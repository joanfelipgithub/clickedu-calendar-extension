\## Installation

1\. Download or clone this repository

2\. Open Chrome and go to `chrome://extensions/`

3\. Enable "Developer mode"

4\. Click "Load unpacked"

5\. Select the extension folder



\## Usage

\- Overlay appears automatically on first login to ClickEdu

\- Press SPACE to toggle overlay anytime

\- Add notes to events - they persist across sessions

\- Click "Download" to save calendar as text file



\## Privacy

\- No data is sent to external servers

\- All notes stored locally on your device

\- Only accesses ClickEdu calendar pages
